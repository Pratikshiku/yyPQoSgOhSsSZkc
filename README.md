## University Freshman Registration System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/299374c795e44faf8ddcff637274f1a7/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Registration Process
5. Campus Announcements
6. Personal Center


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Student Management
4. Registration Process Management
5. Student Registration Management
6. Class Assignment Management
7. Dormitory Assignment Management
8. Payment Information Management
9. Student Information Management
10. Class Information Management
11. Dormitory Information Management
12. System Management (Carousel Management, Campus Announcements)
